# Experiment 18: Statistical and Specialized Data Visualization

## AIM
To study statistical and specialized data visualization techniques using Python libraries.

---

## THEORY

### 1. Data Visualization
It is the graphical representation of data to understand patterns, trends, and relationships easily.

### 2. Matplotlib
A Python library used for creating static, animated, and interactive visualizations.

### 3. Seaborn
Built on Matplotlib, it provides better aesthetics and advanced statistical plots.

### 4. Area Plot
It shows cumulative values over a range and is useful for visualizing trends with filled areas.

### 5. fill_between()
A Matplotlib function used to fill the area between curves or between a curve and axis.

### 6. Multiple Area Plot
Used to compare multiple datasets by plotting filled areas in the same graph.

### 7. sns.set_style()
Used to apply different visual themes like grid or dark background.

### 8. Pie Chart
Displays data as slices of a circle representing percentage contribution.

### 9. autopct
A parameter in pie chart used to display percentage values on slices.

### 10. Donut Chart
A variation of pie chart with a hollow center for better readability.

### 11. plt.Circle()
Used to draw a circle (used here to create donut hole).

### 12. gcf() & gca()
Get current figure and axes for modifying plots.

### 13. Boxplot
Used to visualize distribution of data and detect outliers using quartiles.

### 14. Outliers
Extreme values that lie far from other observations in the dataset.

### 15. Heatmap
Displays correlation matrix using colors to represent strength of relationship.

### 16. corr()
Calculates correlation between numerical variables.

### 17. annot=True
Displays numerical values inside heatmap cells.

### 18. cmap
Defines color scheme of the heatmap.

### 19. Bubble Plot
A scatter plot where size of marker represents a third variable.

### 20. plt.scatter()
Used to plot points on graph with x and y values.

### 21. size (s parameter)
Controls size of bubbles in scatter plot.

### 22. hue
Used in Seaborn to represent additional variable using color.

### 23. palette
Defines color range in Seaborn plots.

### 24. sizes
Controls minimum and maximum size of bubbles.

### 25. Histogram
Displays frequency distribution of continuous data using bins.

### 26. bins
Defines number of intervals in histogram.

### 27. edgecolor
Used to highlight boundaries of histogram bars.

### 28. Colorbar
Used to show scale of colors in plots.
